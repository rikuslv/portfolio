What is a Bandit?

Bandit is a game that, like most other games, is organized in levels. You start at Level 0 and try to “beat” or “finish” it.
Finishing a level results in information on how to start the next level.
The pages on this website for “Level <X>” contain information on how to start level X from the previous level.
